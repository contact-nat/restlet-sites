StackOverFlow

Restlet Framework users have increasingly used this popular web site to post their questions and answer to each other. We encourage you to use it as it tends to provide rapid and good quality answers. The Restlet core team also tries to participate.

  • Search questions tagged 'restlet'.
  • Ask a new question, adding the 'restlet' tag to it.

Mailing lists

This is traditional way used so far by Restlet Framework users and developers to communicate. To participate, just follow these steps:

  1. Create an account on Tigris.org (or login if you have one already),
  2. Subscribe to the "Discuss" mailing list (other lists available),
  3. Post your message using:
    • a mail client,
    • the web site,
    • the newsgroup gateway.

You can also search the archives.

Professional services

As the founder and main supporter of this open source project, Restlet is offering professional services including support plans, co-development, custom development, expert consulting and training.
