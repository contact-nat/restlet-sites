This sample code is a set of three Eclipse projects. There are provided without binaries dependencies.

See each project's readme file for more details about installation and dependencies.

The following binaries are missing and must be added :

org.restlet.example.firstapplication.android project:
 * /libs/android-support-v4.jar                (regular android project jar)
 * /libs/org.restlet.jar                       (core module of Android edition for Restlet Framework)
 * /libs/org.restlet.ext.jackson.jar           (Jackson extension of Android edition for Restlet Framework)
 * /libs/com.fasterxml.jackson.core.jar        (Jackson dependency)
 * /libs/com.fasterxml.jackson.databind.jar    (Jackson dependency)
 * /libs/com.fasterxml.jackson.annotations.jar (Jackson dependency)

org.restlet.example.firstapplication.gae project:
 * /war/WEB-INF/lib/appengine-api-1.0-sdk-1.7.0.jar        (regular GAE project jar)
 * /war/WEB-INF/lib/appengine-api-labs.jar                 (regular GAE project jar)

 * /war/WEB-INF/lib/datanucleus-core-1.1.5.jar             (regular GAE project jar)
 * /war/WEB-INF/lib/datanucleus-jpa-1.1.5.jar              (regular GAE project jar)
 * /war/WEB-INF/lib/geronimo-jpa_3.0_spec-1.1.1.jar        (regular GAE project jar)
